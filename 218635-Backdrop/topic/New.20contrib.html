Some modules I've ported recently: User Prune https://github.com/backdrop-contrib/user_prune (alternative to logintoboggan which hasn't been ported but is partly in core), Link Checker https://github.com/backdrop-contrib/linkchecker, and Webform Localization https://github.com/backdrop-contrib/webform_localization which also required @indigoxela and I working on an upgrade path for i18n and hopefully a release soon. (And a thanks to the client that agreed to a healthy budget for the upgrade.)

backdrop-contrib/user_prune
Delete inactive users in batches based on criteria. - backdrop-contrib/user_prune
backdrop-contrib/linkchecker
Checks for broken links in content. Contribute to backdrop-contrib/linkchecker development by creating an account on GitHub.
backdrop-contrib/webform_localization
Provides multilingual features to the Webform module. - backdrop-contrib/webform_localization
