klonos (Feb 18 2021 at 08:55):

Thanks @cptX ...changing the markup of the current layout templates would break existing sites, and since in Backdrop we take care of backwards compatibility and try not to break people's sites, this will be hard to be changed in the 1.x version. In https://github.com/backdrop/backdrop-issues/issues/4929 some very interesting ideas were thrown on how to best handle such changes in general:

  • create new layout templates, and deprecate (but still support) the existing ones.
  • add new regions to existing layout templates.
  • see if we can fix these sorts of issues with some CSS (for which we have a bigger issue: https://github.com/backdrop/backdrop-issues/issues/4782)
