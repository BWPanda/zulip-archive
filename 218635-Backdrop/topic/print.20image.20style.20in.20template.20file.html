Laryn (Mar 13 2020 at 16:00):

If you have devel installed you should be able to use dpm($item); to see what data is available, and then use theme_image_style to adjust the image style that gets used.

Herb (Mar 14 2020 at 11:26):

There's also a built-in debug command which does the same but dpm formats it nicer. Remember to clear cache with each change

BWPanda (Mar 14 2020 at 12:27):

@mazze.ch I setup a similar template file to yours in my sandbox to play with this, and adding this line produced thumbnails that link to large images:
<a href="<?php print image_style_url('large', $item['#item']['uri']); ?>"><img src="<?php print image_style_url('thumbnail', $item['#item']['uri']); ?>" /></a>
