StPaulTim (Aug 02 2020 at 19:47):

@Olaf Grabienski or anyone. I've never worked with the translation features in BackdropCMS. What is the best place for me to get started? Do we have any videos or documentation on how translations work in Backdrop CMS?

mazze.ch (Aug 03 2020 at 08:20):

@StPaulTim as far as I know, there's no documentation for this, maybe because scenarios are very different (e.g. if Backend also needs to be translated or not). However, getting started is very easy, enabling modules, enabling translation per content type, add language switcher block to the active layout. It's very important to know that each translated node has a different NID (which is why D7 came up with Entity Translation Module (one NID for all language versions). But for getting started, basic content translation works pretty well.
