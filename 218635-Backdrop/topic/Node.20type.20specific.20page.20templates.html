BWPanda (Mar 09 2021 at 02:25):

@Alan Mels I wouldn't do this in a template file, I'd do it in a preprocess function... Try this:

function [MY_THEME]_preprocess_node(&$variables) {
  if ($variables['type'] == 'post') {
    $title = backdrop_get_title() . t(' - Post');
    backdrop_set_title($title);
  }
}
