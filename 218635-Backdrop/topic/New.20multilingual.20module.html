If anyone's interested, there's a brand new contrib module available that can set different frontpages per language. Testing and feedback is welcome.

I think, it fills another internationalization gap, as different start (home, front) pages per language are a common task, even if there's no need for a fully blown internationalized site.

https://github.com/backdrop-contrib/frontpage_multilingual
