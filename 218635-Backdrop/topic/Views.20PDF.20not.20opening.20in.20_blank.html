BWPanda (Mar 31 2020 at 13:13):

I think you need to override theme_file_link in your theme instead of doing this via the Views UI. Reason is that the file is being output as an icon followed by a textual link. When you add your own link using Views, it wraps all of that in another link. Since you can't have a link within a link, then your Views link is being applied to just the icon and it's using this theme function's link for the text. So the only way to override that is via the theme function (or use a different formatter in Views).
