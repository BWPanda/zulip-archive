@BWPanda Do you have a code sample of how you used font awesome module in your theme? I'm interested, but not sure how it would plug in. I've been just uploading the images and adding the css.

BWPanda (Sep 23 2020 at 23:31):

@Elisabeth Garbeil I use SASS and have a mixin for adding Font Awesome icons. The CSS it produces looks like this:

.scroll-down a::before {
    content: "\f107";
    display: inline-block;
    font-style: normal;
    font-variant: normal;
    text-rendering: auto;
    -webkit-font-smoothing: antialiased;
    font-family: "Font Awesome 5 Free";
    font-weight: 900;
}

Does that help?
