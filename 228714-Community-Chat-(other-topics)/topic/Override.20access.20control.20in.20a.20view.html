function module_name_entity_view_mode_alter(&$view_mode, $context) { if ($context['entity_type'] == 'node' && $context['entity']->type == 'my-content-type') { if (user_is_anonymous()) { if (drupal_is_front_page()) { $view_mode = 'teaser'; } } } } }

Phil Ward (Dec 09 2020 at 07:39):

According to my buddy over at ModulesUnraveled:

Disable SQL rewriting (This will disable SQL rewriting by modules that implement hook_query_alter() and node_access checks.)

It basically means "if I have a user field, and the requesting user doesn't have permission to that field, don't allow access." Disabling says "ignore the all access checks and just show it to everyone regardless of their credentials".

Phil Ward (Dec 09 2020 at 07:51):
