Just trying to get some metrics around it, so to move #788 forward if possible. I keep setting the admin bar to sticky on the sandboxes I work on (which reminds me the same thing I used to do with disabling the Overlay module in D7 :rolling_eyes: ), and then also:
a) WordPress has its admin menu on the side, which makes it available at all times; whereas with Backdrop, it is hidden when you scroll down long pages, and then you need to scroll back up to navigate elsewhere. c) I really like https://www.drupal.org/project/gin which also has the admin bar on the side.
