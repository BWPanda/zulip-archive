BWPanda (Mar 25 2020 at 08:30):

A client has asked me to make a slideshow made up of sections: each section will have some text, and some images that transition in the background. Once a section has shown all its images, the next section is displayed. My problem is that I don't know how to do 'section's in a slideshow. As far as I'm aware, slideshows are just a collection of slides, each with a background and optional text. To have the same text stay in place while the background changes is fine if that's the entire slideshow, but this is like having multiple slideshows all together... Anyone done anything like this before?

BWPanda (Mar 25 2020 at 08:31):

And I know I can just create multiple slides with the same text and different background, but then the text will transition with the background which'll look weird IMO...

klonos (Mar 25 2020 at 20:27):

@docwilmot the current implementation of slideshow modules for D7/Backdrop scroll each slide and the respective text as a single item. @BWPanda (Peter) wants to scroll the background images of the slides independently of the text, because some slides have the same text. So my understanding is that the slideshow looks something like this:

  • bg image 1 with text 1
  • bg image 2 with text 1
  • bg image 3 with text 1
  • bg image 4 with text 2
  • bg image 5 with text 2
  • bg image 6 with text 3
  • bg image 7 with text 3
    ...animating images 1, 2, and 3 should NOT animate text 1
    ...animating from image 3 to image 4 should animate from text 1 to text 2
    ...animating images 4 and 5 should NOT animate text 2
    ...animating from image 5 to image 6 should animate from text 2 to text 3
    ...animating images 6 and 7 should NOT animate text 3
    ...animating from image 7 to image 1 should animate from text 3 to text 1
