indigoxela (Sep 09 2020 at 12:59):

... and later on (tomorrow): good morning @BWPanda !

Definitely grid. (The css selectors are just placeholders)

.parentdiv {
display: grid;
grid-template-columns: [left] 50% [right] 50%;
}

.parentdiv > div {
grid-column: left;
}

.parentdiv .thirdblock {
grid-column: right;
grid-row-start: 1;
grid-row-end: 4;
}

Yes, it's possible to name areas using brackets like [left], I'm not sure how good the browser support is, though.
