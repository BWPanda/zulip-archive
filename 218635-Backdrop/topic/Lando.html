klonos (Dec 11 2020 at 20:20):

Here's my one-liner to spin up a D7 site + admin_menu on my local using lando (broken down to multi-line here, for readability):

lando init \
  --source remote \
  --remote-url https://github.com/drupal/drupal/archive/7.x.zip \
  --remote-options="--strip-components 1" \
  --recipe drupal7 \
  --webroot ./drupal-7.x \
  --name d7test \
  && lando start \
  && lando drush si standard \
  --db-url=mysql://drupal7:drupal7@database:3306/drupal7 -r /app/drupal-7.x -y \
  && drush --root=/app/drupal-7.x dl admin_menu -y \
  && drush --root=/app/drupal-7.x en admin_menu admin_menu_toolbar -y
