Eric Vander Wal (Mar 12 2021 at 12:20):

Does backdrop have any support for custom tokens right now?

bugfolder (Mar 12 2021 at 14:26):

Yes. All tokens that don't need context currently work in Token Filter module, and Backdrop has the same support for creating custom tokens that Drupal did; you just implement hook_token_info() and hook_tokens(), both of which are documented in the api. See User Time Zone Tokens for a recent example of creating custom tokens in a module.

bugfolder (Mar 12 2021 at 14:28):

If you install Token Help, you can easily see all the tokens that are currently available. (Token help doesn't add any tokens; it just makes it easier to see the existing ones.)
