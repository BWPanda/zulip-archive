docwilmot (Feb 02 2021 at 20:15):

The problem here is the wrappers in our current core layouts. We have a big container class with a width restriction, and all the rows are in that. The Hero layouts swap this, to have a wrapper that can be overridden to have no margins, and each row has the fixed width container class within the row. It would be a simple change to do this, and then we could as @klonos suggested, just park the current ones in the legacy folder.
