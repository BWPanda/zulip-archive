Phil Ward (Mar 12 2021 at 17:14):

@Eric Vander Wal I'm not one to give a definite answer, but I would think you would add the view to a layout path as opposed to adding a layout to a view path.

If it were me, I would create a new layout with the custom path, (not a wildcard path) then create a view with a block and add that block to the layout.

Laryn (Mar 12 2021 at 17:16):

If you want to apply an existing layout to a View page, you can also test out the "Layout Wildcard" module. It adds a new field to layout configuration for additional paths to apply the layout to (besides the main context) and you can add the View path there.
https://github.com/backdrop-contrib/layout_wildcard

backdrop-contrib/layout_wildcard
Contribute to backdrop-contrib/layout_wildcard development by creating an account on GitHub.
