Olaf Grabienski (Apr 07 2020 at 16:23):

So, here's a (simplified) description:

  • I have an Event content type with Title, Body text, Location, the Event date, and a Registration deadline date. There are several events on the website.
  • On the Event content type there is another field: a node reference autocomplete to webforms. If I select the webform in question (always the same, see below), it will be displayed / embedded on the Event page.
  • The webform is built to allow registering for different Events. The form is always the same, and it doesn't hold a date. It contains only a Name field, contact details, and a checkbox "I want to attend". (There is also a token-based read-only field which holds the Event title so that I can filter the submissions by event.)

Now, when I reference the webform as an editor, it will be embedded on the Event page. Then, at the Registration deadline date, I have to disable the form manually. My goal is to build something which disables the form automatically at the deadline date.
