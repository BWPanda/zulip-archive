@Quinn Dombrowski,
You need the full URI for image fields in the output rows (i.e. hppts://{site-name}/sites/default/files/{file-name} ). There's a drupal module that allows you to export the full URL (https://www.drupal.org/project/image_url_formatter) in views - use rewrite. Someone else here explained to me how to do it exactly and it works great). Once you do that, then the mapping on the input for feeds that you have will work.
