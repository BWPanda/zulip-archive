Olaf Grabienski (Feb 01 2021 at 13:41):

Rabbit Hole works not only for node content but also user accounts and taxonomy. Also, it has more options than the 'Hidden path' setting of Backdrop core.

bugfolder (Feb 01 2021 at 18:05):

In the small number of modules I've ported, I've added a section "Differences from Drupal 7" to the README file when applicable. If a contrib module feature was adopted in core (and so removed from the module), I noted it there, an example being View Unpublished. Might be helpful to put something like that in modules similarly affected, where core now provides part of the functionality but there's a reason for the contrib module to still exist.
