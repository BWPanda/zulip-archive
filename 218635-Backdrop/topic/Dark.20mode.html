indigoxela (Oct 30 2020 at 07:15):

Not sure if/how it's possible to make that dark too...

It's possible to get a dark content edit area via hook_ckeditor_settings_alter() - with something like $settings['contentsCss'][] = $color_css;.
Here's an example (Lateral with the "Night" preset):
ckeditor-dark.png

It's also possible to change css files only via ckeditor_stylesheets[] = css/yourcss.css in your theme's info file.
