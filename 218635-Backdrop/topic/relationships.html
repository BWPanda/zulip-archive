klonos (Apr 05 2020 at 03:28):

@Justin Keiser create a new layout -> set its context to node/% -> head to the block management of that layout -> click "Add block" on any region -> search for blocks with field names, like "body" or "image" ...this means that you can render nodes of type "Post" and have the body in the main content region, while rendering the image in the sidebar :wink: ...let your creativity run wild!
