BWPanda (Feb 21 2021 at 20:07):

How is the list generated actually?

If the user profile has a GitHub username (in the Social field), it uses that. Otherwise it uses the normal username field. It then searches for that username in the Body field of project nodes on B.org and returns matching results.

BWPanda (Feb 21 2021 at 20:10):

Any chance of adding "has submitted a PR on the module" to the inclusion criteria?
