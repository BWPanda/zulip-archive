#skip-link {

  pointer-events: none;
}

  content: "=";
  position: fixed;
  top: 1rem;
  left: 1.25rem;
  color: #f56a6a;
  cursor: pointer;
  font-size: 2rem;
  pointer-events: auto;
  z-index: 999;
}

Hi ajlkn,

Recently I've built a theme for the open source CMS https://backdropcms.org. The theme is inspired by your wonderful Editorial template (html5up.net/editorial). I haven't made an official release yet but you can have a look at the theme at https://github.com/backdrop-contrib/editorial.

I'm not sure if the CC Attribution license applies in my case, as I didn't use the HTML5 UP code. And I've however mentioned Editorial and HMTL5 UP as inspiration at the top of the README.
