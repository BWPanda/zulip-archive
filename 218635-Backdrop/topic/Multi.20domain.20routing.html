StPaulTim (Dec 04 2020 at 04:17):

@Eliza Olivares - I think you are looking for the equivalent of Domain Access Module for Drupal?

Domain Access module allows you to configure multiple domains to a single Drupal site and then select which nodes are used on which domain. It's a very powerful module and I have used it extensively for Drupal 7, but I have not yet found a similar solution for Backdrop CMS.

https://forum.backdropcms.org/forum/domain-access-equivalent
