Elisabeth Garbeil (Mar 26 2020 at 16:19):

Jason,

You can go directly from D6 to Backdrop if you do it as a migration and export the data, then import to Backdrop using feeds. We've done it this way pretty successfully. I've got one in development now that I've had to put on the back burner because it's a nonpaying job, but we had no problems. I'm in the look and feel stage with that one - the export/import went well. I always do migrations because we're typically redesigning / updating the site at the same time. D6 layout/design is pretty outdated now. Also - safer. Just a note, I changed content types in D6 before I exported so that I could streamline. For example - Simplenews newsletters became articles. (I stopped using simplenews so these are back issues).
D6 - node export is the module used. Also, no desire to mess with live production sites while in development.

Elisabeth
