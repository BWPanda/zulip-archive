Mine are fine, but I develop on iOS/Safari, and it's very theme dependent. If you have the debugger, check the location. I've found the best css uses position absolute with a top and right, and at least 2% padding to allow for the scroll bars.

Here's a sample of css from one of my sites (bear in mind that it's very theme-dependent)
.menu-toggle-button {
position: absolute;
top: -8em;
right: 20%;
z-index: 999;
margin-top:1%;
}
