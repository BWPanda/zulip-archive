@BWPanda (Peter) - Can you share how you generated those Gitter stats?

BWPanda (Feb 04 2020 at 21:17):

Certainly! First I downloaded an archive of all Gitter messages for the main Backdrop channel using this archive-gitter Python script. That gave me a 55MB JSON file (which crashed my computer twice; once when I tried to open it in a text editor, and again when I tried using an online JSON -> CSV converter :laughing:) I then researched how to parse a JSON file using PHP without loading the whole thing into memory. I found a bunch of PHP libraries to that effect, but they required Composer to install and I wanted something quick-and-easy. So I wrote my own PHP code that went through the file line-by-line and wrote the necessary data to a new CSV file. See my code here: https://gist.github.com/BWPanda/81c0571d78764b73ad036bdfafb38904 Then it was just a matter of playing around with the data in a spreadsheet to count and sort it all :smiley:
