Herb (Aug 08 2020 at 21:19):

I found a combo that works.

So in hook_autoload_info() the namespace should be there like: 'MyNamespace\Myclass' => 'libraries/mylibrary/src/myclass.php', and then refer to the class like $myinstance = new \MyNamespace\Myclass();

I tried a few different combos including just referencing the class without namespace. I think in the end it was a mistake with the case of the namespace.
