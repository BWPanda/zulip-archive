klonos (Apr 27 2020 at 22:29):

...a bit of more info on Merlin from the main developer involved in its development:

It’s a soft launch as there are still a few things we’re hoping to clean up (namely around docs/onboarding) and a couple bugs to squash. ...Merlin was built first and foremost to help make migrations from legacy CMS (or any website) to Drupal easier. It crawls and scrapes content and creates structured json ready for import via migrate api.

It’s got support for menus, taxonomy, content, paragraphs (incl. reusable paragraphs via library), media, linkit url rewriting, media, and plenty more. It’s also generic by nature, so can be used for migrations to Wordpress, or to create structured data for any other purpose (e.g ecommerce competitive pricing analysis, static site exporter, 404 link checker across sites)..

We’ll aim to launch a blog series of how-to guides that help explain all the use cases, it’s still early days and an evolving tool but hopefully it’ll become a go-to when people think about content migration.
