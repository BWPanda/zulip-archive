klonos (Feb 09 2020 at 18:42):

I ended up doing something like this in the script I use:
BACKDRUSH="$(curl -sL https://api.github.com/repos/backdrop-contrib/drush/releases/latest | jq -r '.tag_name')" ...and then sed -i '' -e "s/'?.?.?'/'${BACKDRUSH}'/g" .lando.yml

klonos (Feb 09 2020 at 18:43):

^^ this grabs the latest tag from the (Backdrop) drush repo, and then sets its value in the .lando.yml file for me.
