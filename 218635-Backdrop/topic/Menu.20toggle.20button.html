Elisabeth Garbeil (Sep 30 2020 at 16:00):

BWPanda said:

When you're theming a site and you need to change when a drop-down menu changes to a menu toggle button (i.e. a hamburger menu), how do you do that?

it's fairly easy. Just use @media with the breakpoint you want and then hide the dropdown menu (display:none) and unhide the menu-toggle. ex:
@media screen and (max-width: 48em) {
#main-menu { display:none; }
#menu-toggle-button {display:block;}
}
The exact syntax and breakpoints depend on your particular theme and use case. I find I prefer to go to the menu toggle at the biggest point it makes sense.
