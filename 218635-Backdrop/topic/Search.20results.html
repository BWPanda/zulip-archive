BWPanda (Apr 29 2020 at 17:51):

@Laryn I've done something similar here: https://panda.id.au/search/node/test
To achieve that, I have this code in my theme's template.php file:

/**
 * Overrides template_preprocess_search_results().
 */
function [THEME]_preprocess_search_results(&$variables) {
  foreach ($variables['search_results'] as &$result) {
    $result['info'] = $result['url'];
  }
}

Maybe that helps you make the changes you need...?
