oadaeh (Jason Flatt) (Feb 16 2021 at 21:14):

And realised that I always assumed that any new Entity would be immediately available to Views, like in D7 via Entity API. Seems though we didnt port that functionality. Should we?

In D7, new entities were never automatically available to Views, even by depending on the Entity API module. Those entities needed to have special elements from the Entity API added to their entity_info() array to integrate with Views.
For example: 'views controller class' => 'EntityDefaultViewsController',
