Laryn (Feb 20 2020 at 18:06):

I'm trying to render the search block from within a custom module. This seems to work, except the values I'm trying to pass as parameters aren't taking (e.g. the button shows up and no placeholder text in the search field). Anyone have an idea what I need to change here?

$searchblock = module_invoke('search', 'block_view', array( 'placeholder' => 'Search', 'button_text' => ''));
  $output = render($searchblock['content']);

Laryn (Feb 20 2020 at 18:21):

Update: First line seems to work like this:
$searchblock = module_invoke('search', 'block_view', 'search_block', array( 'search' => array('placeholder' => 'Search', 'button_text' => '')));

BWPanda (Feb 20 2020 at 19:00):

@Laryn Yep, you got it. Note however that the 3rd parameter to module_invoke() is the $delta of the block, and that search_block_view() doesn't seem to use that parameter at all, so you could make that an empty string, or anything else, and it wouldn't matter.
