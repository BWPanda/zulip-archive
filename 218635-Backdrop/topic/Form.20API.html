The new Form API Reference page is live! https://api.backdropcms.org/form_api This page is built in Backdrop using content types and views (the old page was a single Body field full of hand-coded HTML). This page should be much easier to update now, as most of the heavy-lifting has been automated. If you're an editor on the site, you'll see contextual Edit links next to each element/property. Take a look and let us know (here or in the API issue queue) if there are any bugs.
