BWPanda (Nov 27 2020 at 01:09):

The dragon image is part of the "Powered by Backdrop" block when used in the default theme, Basis. So you can either:

  • Remove the "Powered by Backdrop" block
  • Change the theme to a different one
  • Add some custom CSS to your theme to hide the image
